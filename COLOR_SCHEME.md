## ðŸŒˆ Dynamic Theming

This setup supports dynamic theming. When you use the **Theme Switcher** (`SUPER + CTRL + T`), the following files are updated:

1.  **Hyprland**: `~/.cache/hypr_colors.conf`
2.  **Waybar**: `~/.cache/waybar_colors.css`

The default colors above are used as fallbacks if no wallpaper theme has been generated.
